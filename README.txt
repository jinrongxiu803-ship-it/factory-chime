============================================================
[자동 실행 방식] / Cach tu chay
============================================================
- PC 를 켜면 → 창 없이 조용히 백그라운드 실행
- 설정 시간이 되면 → 차임벨 / 스트레칭 자동 시작
- 우측 하단 트레이의 "종 아이콘" 으로 상주
- 종 아이콘 클릭 → 설정 열기 / 스트레칭 테스트 / 종료

============================================================
[기본 벨 시간] / Gio chuong mac dinh
============================================================
월~금: 07:25, 07:30, 12:00, 12:55, 13:00, 18:00
토   : 07:25, 07:30, 12:00, 12:55, 13:00, 17:00
일   : 휴무 / Nghi
스트레칭: 14:15

→ 시간 변경: 트레이 종 아이콘 클릭 → 설정 창

============================================================
[파일 설명] / Cac file
============================================================
INSTALL.bat        ← 이것만 더블클릭하면 됨 (처음 한 번)
START.bat        ← 평소 수동으로 켤 때 (선택)
chime.py        ← 메인 프로그램
stretching.py   ← 스트레칭 안내
generate_voice.py ← 음성 생성 (INSTALL.bat 이 자동 실행)
school_bell.wav ← 학교 벨소리
relax_music.wav ← 스트레칭 배경음악
voices/         ← 생성된 고음질 음성 (설치 후 자동 생성)

============================================================
[스트레칭 안내] / Huong dan gian co
============================================================
- 오후 졸음/피로 해소 (운동전문가 루틴 기반)
- 차분한 여성 베트남어 고음질 음성 (아주 천천히)
- 긴장 완화 배경음악 + 화면 동작 안내 (베/한)
- 약 6~7분, 14단계
- 여러 시간 설정 가능 (트레이 → 설정 창)

============================================================
[음성/속도 바꾸기] / Doi giong / toc do
============================================================
generate_voice.py 위쪽 수정 후 다시 실행:
  python generate_voice.py
  VOICE = "vi-VN-HoaiMyNeural"  (여성, 기본)
  VOICE = "vi-VN-NamMinhNeural" (남성)
  RATE  = "-25%"  (느림. -40%=더 느림)

============================================================
[참고] / Luu y
============================================================

- 기본 벨 시간 새로 적용: chime_config.json 삭제 후 재실행
